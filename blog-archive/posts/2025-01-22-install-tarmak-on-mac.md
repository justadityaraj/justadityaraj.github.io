# How To Install Tarmak On Mac

I tried installing Tarmak as per the <a href="https://forum.colemak.com/topic/1858-learn-colemak-in-steps-with-the-tarmak-layouts/" class="ek-link">forum</a>, but the mac links were broken. Karabiner has been significantly updated, the old docs didn’t work, couldn't find the Tarmak layouts in Karbiner either.

So I built the layouts in Karabiner myself. If you're familiar with Karabiner, you can access the Tarmak profile and Alfred workflow here, <a href="https://github.com/justadityaraj/tarmak-karbiner-alfred" class="ek-link">github.com/justadityaraj/tarmak-karbiner-alfred</a>. Otherwise, just follow this blog.

## Install Karbiner Elements

<a href="https://karabiner-elements.pqrs.org/" class="ek-link">Download Karbiner</a>, (a keyboard customizer for MacOS) and install it.

### Tarmak Layouts on MacOS

Open it up, then go to Misc \> Export & Import \> Open config folder.

Download <a href="https://github.com/justadityaraj/tarmak-karbiner-alfred/blob/main/karabiner.json" class="ek-link">this karabiner.json</a> file, which has the Tarmak layouts.

Replace the karabiner.json file in the config folder with this one to import the Tarmak profiles.

If you're *already using Karabiner* and have profiles built, copy the code from my karabiner.json file and merge it with your existing one. Also, make sure to back up your current karabiner.json before doing this.

### Tarmak 1

Go to the profiles in Karabiner, and you'll see 6 profiles.

- Default Profile (Qwerty)
- Tarmak 1,2,3,4 and 5

Switch to Tarmak 1 and you're good to go.

## Shortcut using Alfred Workflow

If you use <a href="https://alfred.app/" class="ek-link">Alfred</a>, you can import <a href="https://github.com/justadityaraj/tarmak-karbiner-alfred/blob/main/Tarmak-switcher-karbiner.alfredworkflow" class="ek-link">this workflow</a> that I created to quickly switch the profiles.

The shortcut is **NKJ** (for when you're on the default "Qwerty" profile) and **KEN** (for when you're using any of the Tarmak 1-5 profiles).

Hope this helps!
