# How To Run DeepSeek Offline on Windows or Mac

Download and install <a href="https://www.nomic.ai/gpt4all" class="ek-link">GPT4All</a> (open-source offline chatbot)

Once installed, click on Models (on the left side) \> Add Model \> HuggingFace.

Search for: bullerwins/DeepSeek-V3-GGUF (The <a href="https://huggingface.co/bullerwins/DeepSeek-V3-GGUF" class="ek-link">model</a> size is approximately 41GB, with a quantization of f16).

Hit **Download**.

Once the download is complete, go to **Chats** and select the DeepSeek model.

That's it.
