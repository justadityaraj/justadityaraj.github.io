# How To Install Tarmak On Windows

## Install EPKL

Download **EPiKaL Keyboard Layout program** from this <a href="https://github.com/DreymaR/BigBagKbdTrixPKL/releases/tag/EPKL_1-4-1" class="ek-link">GitHub release page</a> and install it.

Look at the bottom-right dock. You’ll see EPKL there, right-click on it and go to **Layout Settings**.

## Tarmak 1 Layout Configuration

Set it up like this:

1.  Main layout: **Tarmak**
2.  Layout type: VK
3.  Keyboard type: ANS
4.  Variant: **Tm1**
5.  Mods: none

For **3. keyboard type** and **5. mods**, pick what matches your setup. If you have an international-style keyboard that's ortholinear, choose ISO-Orth for type. If your keyboard is curved and angled, select CurlAngle.

Hit **Submit Layout Choice**, then click **OK**, and you're good to go!

## EPKL Shortcuts

You can use **Ctrl + Shift + 3** to stop Tarmak (for example, when practicing on typing websites) and jump into Layout Settings directly using **Ctrl + Shift + 6**.

To switch Tarmak layouts later, repeat the steps and just change **4. Variant** to Tm2, Tm3, Tm4, or Tm5 (which is the Colemak layout).
